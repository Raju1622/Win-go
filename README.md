# InstaIndia - Flutter Web

A beautiful Instagram-inspired social media app built with Flutter for Web featuring:
- User authentication (Login/Signup)
- Home feed with Indian context posts
- User profiles with Indian names
- Search and explore
- Beautiful modern UI/UX with Indian theme
- Fully responsive design for desktop and mobile
- Desktop sidebar navigation & mobile bottom navigation

## Getting Started

1. Install Flutter dependencies:
```bash
flutter pub get
```

2. Run the web app:
```bash
flutter run -d chrome
```

Or build for web:
```bash
flutter build web
```

Then serve the `build/web` directory using any web server.

## Features

- Modern and clean UI
- Fully responsive design (desktop sidebar navigation, mobile bottom navigation)
- Optimized for web browsers
- Smooth animations
- Instagram-like interface
- Centered content layout for better desktop experience
