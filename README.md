# Instagram Clone - Flutter

A beautiful Instagram clone built with Flutter featuring:
- User authentication (Login/Signup)
- Home feed with posts
- User profiles
- Search and explore
- Beautiful UI/UX

## Getting Started

1. Install Flutter dependencies:
```bash
flutter pub get
```

2. Run the app:
```bash
flutter run
```

## Features

- Modern and clean UI
- Responsive design
- Smooth animations
- Instagram-like interface
